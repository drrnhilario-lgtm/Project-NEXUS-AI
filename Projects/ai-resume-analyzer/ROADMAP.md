# NEXUS AI Resume Intelligence Platform Roadmap

This roadmap describes direction, not a delivery guarantee. Privacy, explainability, and offline operation remain default design constraints.

## Completed

- PDF text extraction with controlled invalid, encrypted, and image-only errors
- Alias-aware skill matching, evidence, missing skills, and priority classification
- Rule-based ATS readiness with category explanations
- Resume intelligence, career-gap analysis, recommendations, and learning roadmap
- Modular Streamlit dashboard with visual analytics
- Offline HTML and text report downloads
- Synthetic fixtures, security regression tests, and complete offline smoke testing
- Pinned dependencies, release validator, architecture documentation, and release checklist

## Version 1.0

- Complete manual light, dark, mobile-width, navigation, and download QA
- Validate installation in a clean Python 3.12 environment
- Prepare final screenshots, demo media, release notes, and tagged changelog entry
- Publish the documented limitations and privacy guarantees with the release

## Future Version 1.1

- Expand the tested skill taxonomy for additional role families
- Add configurable local skill dictionaries
- Improve accessibility and keyboard navigation
- Add more CLI and Streamlit integration coverage
- Improve in-memory report composition and release packaging

## Future Version 2.0

- Support additional resume and job-description formats
- Add configurable offline analysis profiles
- Provide richer local comparison and explainability workflows
- Package the application for simpler local deployment
- Consider opt-in local history with explicit consent and deletion controls

## Ideas

- User-maintained role templates stored locally
- Import and export of analysis configuration
- Additional portfolio-oriented learning resources
- Localized guidance and terminology

Ideas require privacy, maintenance, and testing review before they become planned work. OpenAI, cloud persistence, and automatic resume rewriting are not part of the current roadmap.
