# Asset Guide

This folder contains reusable NEXUS AI visual assets. The official architecture artwork is available under `assets/architecture/`. The expected future compact logo path is `assets/nexus-ai-logo.svg`, with a PNG fallback only if needed.

Potential future assets:

- Project logo in accessible light and dark variants
- Repository and social-preview banner
- Locally stored icons when native Streamlit or text labels are insufficient
- Additional architecture or data-flow variants derived from the documented system

Assets should be original or properly licensed, optimized for repository use, and accompanied by source or attribution information when required. Do not add remote dependencies, tracking pixels, personal data, or decorative files without a documented use.
