# Demo Guide

This folder is reserved for a short portfolio demo GIF or video. No demo media is included in the release candidate yet.

## Recommended sequence

Target a **30–60 second** recording:

1. Launch the Streamlit application.
2. Upload a synthetic text-based PDF resume.
3. Paste a synthetic job description.
4. Select **Analyze Resume**.
5. Show Dashboard, Results, ATS Review, and Learning Roadmap.
6. Open the Download Center and demonstrate HTML or text download.

## Production requirements

- Use synthetic data only.
- Hide local usernames, paths, notifications, browser bookmarks, and unrelated tabs.
- Do not display API keys or personal information.
- Keep motion readable and avoid excessive zooming.
- Optimize the final file for GitHub viewing and provide captions or accompanying text.
