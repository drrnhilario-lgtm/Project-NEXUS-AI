# Screenshot Guide

Store polished Version 1.0 portfolio screenshots here. Use synthetic resume and job-description data only, crop out local usernames and file paths, and verify that no personal contact details or API credentials are visible.

Recommended screenshots:

1. Dashboard — executive metrics, insight, and visual analytics
2. ATS Review — score, category breakdown, strengths, and limitations
3. Resume Intelligence — readiness summary, career gap, and priorities
4. Learning Roadmap — all three recommended phases
5. Downloads — available HTML/text reports and planned PDF state
6. Settings — offline privacy and functional display preference

Prefer consistent desktop dimensions plus one narrow responsive view. Optimize images before committing and use descriptive lowercase filenames such as `dashboard-overview.png`.
